Tags

Tags are hidden in the front end. I use them for filtering:

  • Front page shows newest blog post tagged with 'broadcast'
  • I submit to newsfeeds only posts tagged with 'broadcast'

Upgrade wordpress

Use the wordpress-automatic-upgrade plugin and follow all the instructions. Important: Do the one step that asks you to click link that opens new window to do DB updates. The link to start the update process shows up right below the WP notice that says there is a new version.
